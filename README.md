# 煤矿井下数字孪生系统

## 项目简介
本项目实现煤矿井下设备数字孪生系统，通过三维可视化技术实时展示设备运行状态。

## 技术栈
- Three.js
- Node.js
- WebSocket
- JavaScript

## 系统功能
1. 三维设备可视化
2. 设备实时状态同步
3. 设备远程控制
4. 设备状态颜色显示
5. 报警闪烁
6. 鼠标点击查看设备信息
7. 三维视角控制

### 需求分析
#### 1. 引言
本系统旨在构建煤矿井下生产环境的数字孪生平台，实现对井下设备、人员、环境的实时感知、可视化与智能分析。
#### 2. 总体描述
面向矿山调度中心，以井下传感器网络（瓦斯、温度、设备状态）为运行环境实现工业级实时监控、决策支持系统。
#### 3.功能需求
(1)三维数字孪生模拟：系统应支持井下巷道、设备的三维建模与展示。输出：glb模型
(2) 实时数据接入与同步：系应统接入井下传感器数据并实时更新。数据类型包括：瓦斯浓度、温湿度、风速和设备运行速度。刷新频率 <= 1s
(3) 安全预警系统：当环境指标异常时发出警告，例如：瓦斯大于阀值报警。输出：声光报警和UI提示
(4) 设备监控与预测维护：监控设备运行状态并预测故障。
#### 4. 数据类型
(1) 实时数据（传感器数据）
(2)静态数据（巷道模型）
(3)历史数据（报警记录）
#### 5. 系统接口 
##### 设备接口
包括设备注册、设备状态上报、设备控制接口
##### 前端实时接口
包括前端注册、初始化设备列表、设备列表更新、设备数据推送
##### REST API 接口
包括获取设备列表、获取设备历史数据、设备详情、报警记录
##### 鉴权接口
包括登录接口、Token
